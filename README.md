# Air Defense Systems – WPF Client

This project was developed as part of the **TEKNOFEST Air Defense Systems (HSS) Competition**.

It is a desktop client and user interface application built using **C#** and **WPF**, focusing on client-side operations and system interaction.

## Technologies Used
- C#
- WPF

## Project Purpose
The main goal of this project is to provide a user-friendly and modular client interface for air defense system simulations and operations.

## Notes
This project was developed for competition and educational purposes.
