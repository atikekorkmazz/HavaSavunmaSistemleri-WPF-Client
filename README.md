# Air Defense Systems – WPF Client

Bu proje, **TEKNOFEST Hava Savunma Sistemleri (HSS)** yarışması kapsamında geliştirilmiştir.

C# ve WPF kullanılarak geliştirilmiş bir masaüstü client / arayüz uygulamasıdır.

## Kullanılan Teknolojiler
- C#
- WPF

## Amaç
Hava savunma sistemleri için kullanıcı arayüzü ve client-side işlemleri gerçekleştirmek.
